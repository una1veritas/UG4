

**使い方**************************************************

*可視化対象の設定****************************
　１：ファイルシステムでディレクトリ指定。
　２：ドラッグ・ドロップでディレクトリの指定。
　３：ドラッグ・ドロップでファイルの追加。

*下部操作パネル*******************************
　１：REDRAW
	選択クラス・メソッドだけで再描画。
	未選択時は、全てのメソッド再描画。

　２：TRANS / SELECT
	TRANS　	グラフの表示位置平行移動。	
　	SELECT　	マウスによる（ドラッグ）メソッドの選択。
　
　３：CLASS / METHOD
	CLASS		クラスノード描画。
	METHOD	メソッドノード描画。

　４：LOOP BIAS
	ノードの位置に対して、どの程度ループ構造を反映させるか。

　５：CHANGE DIRECTORY
	ファイルシステム起動。対象ディレクトリ指定。
